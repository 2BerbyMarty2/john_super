## john_super

A themed **JavaFX** management system built with **Maven** for tracking dealers and inventory items[cite: 1].

### Features
* **Inventory Management**: Track items and dealers using file-based storage (`Items.txt`, `Dealers.txt`)[cite: 1].
* **Data Logic**: Includes custom `BubbleSort` implementations and input validation[cite: 1].
* **Themed UI**: Fantasy-themed interface styled with FXML and CSS, featuring custom magical assets[cite: 1].
* **Testing**: Comprehensive **JUnit** test suite for storage and management logic[cite: 1].

### Tech Stack
* **Language**: Java[cite: 1]
* **Framework**: JavaFX[cite: 1]
* **Build Tool**: Maven[cite: 1]
* **Testing**: JUnit[cite: 1]
